# Neurocanopy

A Hebrew (RTL) clinical neurofeedback / cognitive-rehabilitation web app prototype.

- **`index.html`** — clinician app (login, dashboard, patients, patient file, protocol builder, research/knowledge hub, settings, real-time session, training & session summaries).
- **`patient.html`** — patient app (welcome + cognitive exercises: growth sequence, missing part, remember the plant).

Self-contained static HTML (all CSS, JS and images inline; only Google Fonts is loaded from a CDN).

University interactive-design course project.
